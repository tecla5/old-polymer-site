# install
> git clone https://github.com/tecla5/site.git

> npm install

> bower install



## generate distribution
> grunt (default: grunt build)

> grunt build

## running the app mode dev or dist
> grunt serve

> grunt serve:dist

http://localhost:9000/




## develop
